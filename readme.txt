== Description ==

*  This plugin lets your customer Choose from all available coupons on your shop . This plugin will simply add extra button on your cart/checkout page that shows nice popup window that will help customer to choose any one coupon from all available coupons. 

* Along with Coupon Select , It will also show other coupon details like expiry date(if set), free shipping(if set yes) , minimum spend(if set) ,discount etc . 

* Radio select will not be available to select in case Coupon is not valid . 





== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `woocommerce-coupon-selector` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
